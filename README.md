# 🎭 playwright-pytest-sandbox
 
A practice repository for learning and experimenting with end-to-end UI test automation using **Playwright** and **pytest**.
 
---
 
## 📌 About
 
This repo contains a collection of autotests written against a demo/target website. It's designed as a hands-on sandbox to explore Playwright's capabilities within the pytest ecosystem — page interactions, assertions, fixtures, and more.

---
 
## ⚙️ Requirements
 
- Python 3.9+
- [pytest](https://docs.pytest.org/)
- [pytest-playwright](https://playwright.dev/python/docs/intro)
